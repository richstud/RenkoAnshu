## Troubleshooting

### Chart still slow?
- Verify MT5 is running and connected
- Check `/api/tickers` returns data
- Restart backend if needed

### Bid/Ask not showing?
- MT5 needs quote data for that symbol
- Try EURUSD or BTCUSD first
- Check browser console for errors

### Backend disconnecting?
- Rare now, but check system resources
- Restart backend if it happens
- Monitor backend logs

## Advanced Config (Optional)

If you want to tune performance:

**File:** `backend/api/renko_chart.py`

```python
# Line 30: Thread pool size (increase for more concurrent requests)
executor = ThreadPoolExecutor(max_workers=4)  # or 8

# Line 135: 1-min candle count (lower = faster)
candle_count = 100  # was 500, now 100

# Line 113: Cache duration (lower = fresher data)
if current_time - cache_ttl[cache_key] < 1.0:  # was 1.0 second
```
