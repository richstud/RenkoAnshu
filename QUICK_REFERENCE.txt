┌─ HOW IT WORKS ───────────────────────────────────────────────────────────┐
│                                                                            │
│  1. User clicks "Enable Auto-Trading" for XAUUSD                         │
│  2. Entry added to auto_trading_watchlist table                          │
│  3. Every 1 minute, auto-trader checks the watchlist                     │
│  4. Gets Renko chart data from MT5                                       │
│  5. Evaluates if brick color changed (green=BUY, red=SELL)              │
│  6. If signal detected, executes trade on MT5                            │
│  7. Stores position in auto_trading_positions                            │
│  8. When closed, moves to auto_trading_history with P&L                  │
│  9. Frontend fetches and displays trades                                 │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘
